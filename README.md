Carousel Slider

Невеликий адаптивний карусельний слайдер, реалізований на чистому JavaScript без використання сторонніх бібліотек.

Можливості

автоматичне перемикання слайдів (autoplay)

кнопка Play / Pause

навігація стрілками

індикатори активного слайду

керування з клавіатури (← →, пробіл)

підтримка свайпів (миша та touch)

плавні анімації без стрибків контенту

Технології

HTML5

CSS3 (Flexbox, transitions)

JavaScript
